1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore file is a text file placed in your git repository that tells git not to track certain files and folders that you don't want being uploaded to your master repository.
.gitignore파일이란 Git 버전 관리에서 제외할 파일 목록을 지정하는 파일이다. git으로 프로젝트를 관리할 때, 그 프로젝트 안의 특정파일들은 Git으로 관리할 필요가 없는 경우가 있다.
