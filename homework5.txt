1. What is .gitignore? You can write the answer either in Korean or English.
git에 올리고 싶지 않거나 올려서는 안되는 파일 or 폴더들을 .gitignore에 기록하면 git의 추적을 피할 수 있다. 이미 git에 등록되어있는 파일이나 폴더는 삭제 후 .gitignore에 등록해야 한다.

2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key
github을 사용할 때는 본인의 개발 내용에 접근하기 때문에 보안이 중요하다. 본인의 github에 접근하기 위해서는 username과 password 또는 ssh key pair를 이용해 본인을 인증해야 하는 경우가 있다. ssh key pair를 사용하면username과 password 없이도 github에 접근할 수 있다.
